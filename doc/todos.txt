- Features
  - CustomShapes mit austauschbaren Hintergrundbildern => Bilder mit runden Ecken, Bilder in Sternen etc
  - Import in eine neue Page: neuer Pagename angeben + Mastervorlage
  - ggf. Import von Tabellen ausschließlich als neue Tabelle mit Import-Tabellennamen oder Angabe neuer Tabellenname
  - später ggf. docx unterstützen
- Bekannte Probleme:
  - Prüfen ob Vectorgrafiken als Bilder verwendet werden können
- Optimierungsmöglichkeiten:
  - getPlaceholderData sollte bei allem Handlern gecached werden (StringOutputModule)
  - Cache für Paar-Knoten (For-EndFor, If-EndIf) // reduziert das Scannen zukünftiger Knoten)
