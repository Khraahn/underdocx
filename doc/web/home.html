Welcome to
Underdocx is an extendable open source framework to manipulate ODT Documents (LibreOffice / OpenOffice). It is a template engine and uses different types of placeholders that can be found and replaced by custom texts, images, tables and other documents. Also documents can be converted to PDF if LibreOffice has been installed.
